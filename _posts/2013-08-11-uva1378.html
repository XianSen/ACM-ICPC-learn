A Funny Stone Game

UVA1378题解

Downloads View on GitHub

题目大意

给定n堆石子(编号从0到n-1),两个人轮流做以下的操作,不能操作的人输

  1. 选三堆石子,我们设他们的编号分别为i, j, k,(i<j <= k)
  2. 从第i堆中拿走一个石子,给j和k分别添加1个石子
给定n(n < 24),和各堆石子数量si(si < 1000)的情况下,求先手是否能获胜

题目分析

实际上我们把题目中的石子看成一堆石子。 具体的说

  • 如果石子在i堆，我们就可以把他看成是有(n - i - 1)的一堆石子。
  • 题目中的操作就转化为,把个数为s的石子转化为更小的两堆石子. 且每堆石子成为了独立的游戏.
  • 这样一次操作就是把一个规模大的游戏,转化为相互独立的两个规模较小的游戏.
如果我们用f(n)表示规模为n的石子的sg,那么把n,分解为n1,n2的两堆, 根据SG定理,其sg既为 f(n1) xor f(n2) 则f(n)即为

 f(n) = mex{f(n1) xor f(n2) n1<n,n2<n,n1 +n2=2 }
